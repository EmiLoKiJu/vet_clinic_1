## Getting Started

This repository includes files with plain SQL that can be used to recreate a database:

- Use [schema.sql](./schema.sql) to create all tables.
- Use [data.sql](./data.sql) to populate tables with sample data.
- Check [queries.sql](./queries.sql) for examples of queries that can be run on a newly created database. **Important note: this file might include queries that make changes in the database (e.g., remove records). Use them responsibly!**

### Install

### Usage

To run the files, you can create a database (or open an existing one), select it, right click and select "Query tool"
After that, you can copy the content of the file you would like to execute in the blank page, and then press F5 to execute it.
For the schema, it will create a table
For the data, it will insert data to the table
For the query, it will show data based on the filters shown in the code.
